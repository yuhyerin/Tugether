  - 글작성 버튼 클릭하면 글작성 페이지로 이동

  <본문>  내가 쓴 글목록 / 스크랩 목록

- [ ] 글작성 페이지

- [ ] 댓글 페이지

- [ ] 알림 페이지 (2개의 탭으로 구성)

  - 요청 탭 ( 나한테 팔로우 요청한것을 승인/거절 하는 요청, )

- [ ] 통계 페이지
